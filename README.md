# 🎮 Gaming Hub

**Gaming Hub** es un proyecto independiente que reúne múltiples juegos y herramientas interactivas en un solo lugar.
La idea es crear un pequeño **centro de entretenimiento web** donde los usuarios puedan abrir juegos clásicos, experimentos interactivos y utilidades creativas sin instalar nada.

El proyecto está diseñado para ser simple, rápido y accesible desde el navegador.

---

## 🌟 Características

* Acceso rápido a varios juegos desde una sola página.
* Interfaz simple tipo **launcher**.
* Funciona directamente en el navegador.
* Incluye juegos clásicos, simulaciones y herramientas creativas.

---
